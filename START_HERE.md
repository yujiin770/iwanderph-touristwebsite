## 3 Things To Do Right Now

### 1️⃣ Create Supabase Storage Buckets (2 min)

Go to https://supabase.com → Your Project → Storage

**Create 3 buckets:**
```
✅ destinations
✅ gallery  
✅ hero
```

**Make them public:**
- Click each bucket
- Click "Policies"
- Make it public (allow SELECT for everyone)

### 2️⃣ Restart Your Servers (1 min)

**Terminal 1:**
```bash
cd backend
npm install
npm run dev
```

**Terminal 2:**
```bash
cd frontend
npm run dev
```

### 3️⃣ Test It! (2 min)

1. Go to http://localhost:5173/admin
2. Click **Gallery**
3. Upload a photo
4. Done! 🎉

## 🆘 Stuck?

**Error: Bucket not found**
→ You didn't create buckets yet (see Step 1)

**Error: 401 Unauthorized**  
→ Make sure you're logged in

**Admin page blank**
→ Clear cache (Ctrl+Shift+Delete)

**Photo not uploading**
→ Check Supabase bucket is public

**Everything broken**
→ Restart both servers
